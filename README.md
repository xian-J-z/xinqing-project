# 大学生心理健康微信小程序 - 心灵伴侣

一个基于微信云开发的大学生心理健康服务小程序，提供心理测评、AI分析、心理咨询预约、心理树洞等功能。

## 功能特性

### 📱 主要功能

1. **登录页**
   - 微信授权登录
   - 获取用户基本信息
   - 应用介绍和功能展示

2. **首页**
   - 用户欢迎界面
   - 轮播图展示
   - 快捷功能入口（心理测评、AI分析、心理树洞、预约咨询）
   - 心理健康文章列表
   - 心理援助热线提示

3. **心理测评**
   - SCL-90症状自评量表
   - 焦虑自评量表(SAS)
   - 抑郁自评量表(SDS)
   - 自尊量表(SES)
   - 测试结果分析和建议
   - 测评记录保存

4. **心理树洞**
   - 匿名发布心声
   - 点赞互动
   - 评论交流
   - 我的发布列表

5. **心理咨询预约**
   - 咨询师列表展示
   - 预约时间段选择
   - 预约原因填写
   - 预约记录管理

6. **AI心理分析**
   - 智能对话交互
   - 情绪状态分析
   - 个性化建议
   - 分析记录保存

7. **个人中心**
   - 用户信息展示
   - 数据统计概览
   - 各类记录查看
   - 退出登录

## 技术栈

- **前端框架**: 微信小程序原生框架
- **云开发**: 微信云开发 (CloudBase)
- **云函数**: 云函数 (Serverless)
- **数据库**: 云开发数据库 (MongoDB)
- **存储**: 云存储

## 快速开始

### 1. 环境准备

- 微信开发者工具 (最新版)
- 微信小程序账号
- 开通云开发服务

### 2. 配置步骤

1. 克隆项目到本地
2. 使用微信开发者工具打开项目
3. 在 `app.js` 中配置云开发环境 ID
4. 上传云函数 `login`

### 3. 运行项目

1. 在微信开发者工具中点击"编译"
2. 使用微信扫描二维码预览

## 数据存储

### 本地存储
- `userInfo`: 用户登录信息

- `aiRecords`: AI分析记录

### 云数据库（需扩展）

- `counselors`: 咨询师信息
- `users`: 用户信息
